# AG-Credits
A credits scene and script for Godot Engine

### How To Use:
* Clone the repo
* Copy "CREDITS" folder to your project folder

You should have this structure:
```
res://
|->CREDITS
   |->GodotCredits.gd
   |->GodotCredits.tscn
```
